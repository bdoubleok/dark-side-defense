# 🏈 Dark Side Defense - CFB 26 Playbook

**Super Bowl LX Champions • 14-3 Regular Season • #1 Scoring Defense**

An interactive web-based playbook for the Seattle Seahawks' 2025 championship defense, adapted for NCAA Football 26 dynasty mode.

---

## 🎯 Features

- **12 Situation Tabs** - Complete coverage for every down and distance
- **75+ Plays** - Comprehensive playbook with proper NFL ratios
- **9 Personnel Packages** - From Dime 1-5-5 to 4-3 Over
- **11 Coverage Concepts** - From Cover 3 Match to Quarter-Quarter-Half
- **Smart Hash Filter** - Formation-aware play recommendations
- **Pressure Decision Tree** - Interactive blitz decision-making
- **Success Tracker** - Log and track pressure effectiveness
- **Formation Filter** - Filter plays by defensive formation
- **DL Alignment Guide** - Detailed alignment for every play

## 🎨 Core Philosophy

### Coverage Distribution (NFL Accurate):
- **Cover 3 Match:** 30% (PRIMARY)
- **Cover 6:** 22% (Split-field emphasis)
- **Cover 4 Quarters/Palms:** 18%
- **Tampa 2:** 13%
- **Other:** 17% (Cloud, Buzz, Robber, Solo, Man, etc.)

### Personnel Usage:
- **Nickel (2-4):** 87% (HIGHEST IN NFL)
- **3-3-5 Over Flex:** Speed package
- **4-3 Over/Under:** vs Heavy only
- **Dime 1-5-5:** vs Empty/5-wide
- **Nickel 3-3 Mint:** 8 in box

### Blitz Philosophy:
- **Target:** 15-20% (2025 actual: 19%)
- **4-Man Rush:** 79-80% (top-5 pressure rate)
- **Stunt Rate:** 21.5% (5th in NFL)

## 🎮 CFB 26 Dynasty Sim Rules

### ✅ ALLOWED:
- 4-man rush 75%+
- Zone majority (60%+)
- Pre-snap shifts and disguises
- Man coverage 20-25%
- Situational blitz 15-20% max
- User MLB or Safety
- Formation adjustments

### ❌ NOT ALLOWED:
- Blitzing 40%+
- User rushing DB/LB every play
- Nano blitzes
- Same play spam
- Only Cover 1 Man all game
